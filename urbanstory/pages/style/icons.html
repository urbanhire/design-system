Urbanicon 1.0

Usage

To be able to use these icons, you must include this line in the portion of your HTML code

<link rel="stylesheet" type="text/css" href="https://uh-font-cdn.urbanhire.com/uh-font-staging.min.css">

To use these icons, use the material-icons class on an element and provide the ligature as the text content.

Call icon with class
<i class="icon-activity"></i>

Call icon with character
<i data-icon="&#xe000;" class="icon"></i>

Use in pseudo class (CSS)
:before{
   content: "\e000";
}
